# Poetry App

A full-stack poetry management application with AI narration.

## Project Structure
```
Kavya/
├── backend/         # Node.js + Express + MongoDB
└── frontend/        # React + Vite
```

## Features

- User authentication
- Create, edit, delete poems
- Search and tags
- Analytics (views, likes)
- AI narration with background music
- Public sharing with comments
- Slideshow view

## Tech Stack

**Backend:**
- Node.js + Express
- MongoDB + Mongoose
- JWT authentication
- REST API

**Frontend:**
- React + Vite
- React Router
- Axios for API calls

## Getting Started

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
